O que foi desenvolvido

Desenvolvido durante a Imersão Dev Alura + Google, este projeto centraliza informações detalhadas sobre os títulos finalistas do Booker Prizes de 2015 até o presente. A aplicação resolve uma lacuna de dados, oferecendo informações que não constam no site oficial, como contagem de páginas, editoras e disponibilidade em português.

O projeto alcançou um marco significativo ao ser selecionado entre os 50 melhores projetos de todo o evento, destacando-se pela utilidade prática e execução técnica.

Aprendizados e Técnicas

A construção desta ferramenta permitiu explorar a fundo a integração entre inteligência artificial e lógica de interface:

  • Integração com Google Gemini (IA): Utilização estratégica de IA para o enriquecimento e estruturação da base de dados literária.
  • Sistema de Busca Inteligente: Implementação de algoritmos em JavaScript que permitem buscas dinâmicas por título, autor ou ano de premiação.
  • Arquitetura de Dados Customizada: Criação de uma estrutura de dados robusta para comportar informações exclusivas de mercado editorial.
  • UX para Curadoria Digital: Foco em uma interface limpa que prioriza a descoberta e o consumo de informações bibliográficas densas.
