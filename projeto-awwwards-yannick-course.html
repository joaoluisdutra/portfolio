O que foi desenvolvido

Neste projeto, desenvolvi um site dinâmico e totalmente responsivo construído do zero (vanilla stack), como parte da certificação da Awwwards Academy sob orientação de Yannick Gregoire. A aplicação foi projetada para ir além do layout estático, focando na criação de uma interface moderna onde o design e a funcionalidade se encontram através de animações fluidas e componentes interativos.

O resultado é uma experiência de usuário (UX) elevada, onde cada elemento da interface reage ao comportamento do visitante, seguindo os mais altos padrões de excelência visual exigidos pela indústria criativa.

Aprendizados e Técnicas

O desenvolvimento deste projeto foi um marco para consolidar o domínio das tecnologias nativas da web e a lógica de programação aplicada à interface:

  • HTML & CSS Avançado: Implementação de estruturas semânticas e estilização complexa, utilizando técnicas modernas para garantir que o design se adapte perfeitamente a qualquer dispositivo.
  • Manipulação do DOM com JavaScript: Foco na criação de componentes "vivos", utilizando JS para gerenciar estados da interface e disparar eventos baseados na interação do usuário.
  • Micro-interações e Animações: Desenvolvimento de fluxos visuais que guiam a atenção do usuário, priorizando a fluidez e a performance do navegador.
  • Padrões Awwwards: Alinhamento com as tendências globais de design de interação, focando em detalhes que transformam um site comum em uma experiência digital memorável.
